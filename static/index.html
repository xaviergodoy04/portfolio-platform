🏦 Interactive Brokers — Flex Query

Para conectar tu portfolio de IBKR sin necesidad de tener TWS abierto:

  1. Entrá a Client Portal → Reports → Flex Queries
  2. Creá una nueva query de tipo Activity Statement
  3. En "Positions", activá: Symbol, Quantity, Cost Basis Price, Mark Price, Position Value, Unrealized P&L, Currency, Asset Class
  4. Guardá y generá un Token (válido 1 año)
  5. Copiá el Token y el Query ID en config.py
💡 Costo: Completamente gratuito para clientes de IB.
🤖 Claude AI — Anthropic

Para el análisis con IA y el chat:

  1. Creá una cuenta en console.anthropic.com
  2. Generá una API key en API Keys
  3. Copiala en config.py → ANTHROPIC_API_KEY
💡 Costo: Claude Haiku es ~$0.001 por análisis. Anthropic da créditos iniciales.

# 1. Instalar dependencias

# 3. Correr
python app.py

http://localhost:5000
📂 Alternativa: CSV Manual

Si preferís no usar la API de Flex Query, podés exportar el Activity Statement desde IB y subirlo manualmente:

  1. En IB → Reports → Activity Statements
  2. Elegí período "Daily" y formato CSV
  3. Descargá y subí el archivo con el botón Subir CSV de esta página (o desde Hoy → Posiciones)
💡 Sin API key. Los precios se actualizan automáticamente desde Yahoo Finance.
